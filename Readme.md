# 🛒 SKU Rationalization Dashboard

Интерактивное приложение для анализа и классификации ассортимента розничной сети (~5000 SKU).

## 🎯 Возможности

- 📊 Визуализация ключевых метрик: маржа, оборачиваемость, волатильность спроса
- 🤖 ML-кластеризация (K-Means / DBSCAN) с бизнес-интерпретацией кластеров
- 📋 Интерактивная таблица решений с возможностью ручной корректировки
- ⚙️ Настраиваемые бизнес-правила для автоматических рекомендаций
- 📥 Экспорт результатов в CSV / Markdown-отчёт

## 🚀 Быстрый старт

### Локально (Python 3.10+)

```bash
# Клонировать репозиторий
git clone https://github.com/serge3emskov/sku_rationalization.git
cd sku_rationalization

# Установить зависимости
pip install -r requirements.txt

# Запустить приложение
streamlit run app.py
